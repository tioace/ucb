CSS Básico: Formatação de Texto e Blocos Pré-formatados

Formatação de Texto com CSS

O CSS nos permite controlar a aparência do texto. Veja as principais tags de formatação:

Texto em negrito usando <b> - apenas altera a aparência visual

Texto importante usando <strong> - indica importância semântica

Texto em itálico usando <i> - para voz alternativa ou humor

Texto com ênfase usando <em> - para ênfase semântica

Texto com destaque especial usando <mark> - como um marca-texto

Texto com tamanho reduzido usando <small> - para notas e avisos legais

Texto que foi removido usando <del> e texto que foi inserido usando <ins> - para controle de versões

Fórmula química da água: H2O usando <sub> - subscrito para fórmulas

Teorema de Pitágoras: a2 + b2 = c2 usando <sup> - sobrescrito para matemática

Blocos de Código Pré-formatados

A tag <pre> preserva espaços, quebras de linha e tabulações exatamente como foram escritos no código fonte.

Exemplo de CSS Básico:

/* Estilos para títulos */
h1 {
    color: #2c3e50;
    font-size: 2.5em;
    text-align: center;
    margin-bottom: 20px;
}

/* Estilos para parágrafos */
p {
    color: #34495e;
    font-size: 1.1em;
    line-height: 1.6;
    margin: 15px 0;
}

/* Estilos para links */
a {
    color: #3498db;
    text-decoration: none;
    transition: color 0.3s ease;
}

a:hover {
    color: #2980b9;
    text-decoration: underline;
}

Estrutura HTML com CSS:

<!DOCTYPE html>
<html>
<head>
    <style>
        .container {
            max-width: 800px;
            margin: 0 auto;
            padding: 20px;
            background: #f8f9fa;
        }
        
        .destaque {
            background: #fff3cd;
            border-left: 4px solid #ffc107;
            padding: 15px;
            margin: 10px 0;
        }
    </style>
</head>
<body>
    <div class="container">
        <div class="destaque">
            Este é um bloco de destaque estilizado com CSS!
        </div>
    </div>
</body>
</html>

Diferenças Semânticas Importantes

<b> vs <strong>

Texto em negrito - Apenas visual

Texto importante - Semântico (leitura em voz alta)

<i> vs <em>

Texto em itálico - Apenas visual

Texto com ênfase - Semântico (alteração de tom)

Dica: Prefira <strong> e <em> para melhor acessibilidade e SEO!
