HTML Básico: Estrutura, Títulos, Parágrafos e Comentários

Hierarquia de Títulos HTML

O HTML oferece 6 níveis de cabeçalho, de <h1> a <h6>, que ajudam a estruturar o conteúdo da sua página de forma semântica.

<h1> - Título Principal da Página

<h2> - Seções Principais

<h3> - Subseções

<h4> - Tópicos dentro das Subseções

<h5> - Detalhes Específicos
<h6> - Informações Menos Relevantes

Dica: Use apenas um <h1> por página para otimização SEO!

Comentários em HTML

Comentários são trechos de código ignorados pelo navegador, mas muito úteis para desenvolvedores.

Uso dos comentários:

  • Explicar trechos complexos de código
  • Desativar temporariamente partes do HTML
  • Devar notas para outros desenvolvedores
  • Organizar seções do código

Estrutura Básica de um Documento HTML

Toda página HTML segue uma estrutura fundamental:

<!DOCTYPE html>
<html lang="pt-br">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Título da Página</title>
</head>
<body>
    <h1>Meu Primeiro Site</h1>
    <p>Este é um parágrafo de exemplo.</p>
</body>
</html>

Elementos essenciais:

  • <!DOCTYPE html> - Define o tipo de documento
  • <html> - Elemento raiz de toda página
  • <head> - Contém metadados e links
  • <body> - Contém o conteúdo visível
