JavaScript: Citações, Listas e Tabelas

Citações em Documentação

"JavaScript é uma linguagem de script orientada a objetos e multiplataforma. É uma linguagem pequena e leve. Dentro de um ambiente de host (por exemplo, um navegador web), o JavaScript pode ser conectado aos objetos de seu ambiente para fornecer controle programático sobre eles."

- MDN Web Docs, Mozilla Foundation

Como Brendan Eich, criador do JavaScript, disse: JavaScript sempre foi sobre dar poder às pessoas para construir coisas na web.

"A linguagem JavaScript foi introduzida em 1995 como uma forma de adicionar programas às páginas web no navegador Netscape Navigator. A linguagem não possuía nenhuma relação com Java, além de uma sintaxe superficial."

- Marijn Haverbeke, "Eloquent JavaScript"

Listas de Comandos JavaScript

Tipos de Dados Primitivos (Lista Não Ordenada)

  • string - Texto e caracteres
  • number - Números inteiros e decimais
  • boolean - Valores verdadeiro ou falso
  • undefined - Valor não definido
  • null - Valor nulo intencional
  • symbol - Valor único e imutável

Passos para Criar uma Função (Lista Ordenada)

  1. Use a palavra-chave function
  2. Defina um nome descritivo para a função
  3. Adicione parênteses para os parâmetros
  4. Inclua chaves {} para o corpo da função
  5. Escreva o código dentro das chaves
  6. Use return para retornar valores (opcional)

Métodos de Array Comuns

  • .push() - Adiciona elemento no final
  • .pop() - Remove último elemento
  • .shift() - Remove primeiro elemento
  • .unshift() - Adiciona elemento no início
  • .forEach() - Executa função para cada item

Tabelas de Referência JavaScript

Operadores JavaScript
Operador Descrição Exemplo
+ Adição ou concatenação 5 + 3 // 8
- Subtração 10 - 4 // 6
* Multiplicação 3 * 4 // 12
/ Divisão 15 / 3 // 5
=== Igualdade estrita 5 === '5' // false

Métodos de String
Método Descrição Retorno
.length Comprimento da string Número
.toUpperCase() Converte para maiúsculas String
.toLowerCase() Converte para minúsculas String
.includes() Verifica se contém texto Boolean
.slice() Extrai parte da string String
